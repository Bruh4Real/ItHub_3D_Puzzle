Для взаимодействия с игрой необходимо использовать мышь и клавиатуру
Чтобы выбрать куб, нужно кликнуть на него мышью
Для поворота куба используйте клавиши W, S, Q, E, A и D. Каждая из них отвечает за поворот куба по соответствующей оси
Для перемещения кубов выбирете один, затем другой с которым необходимо поменять местами
